기능 데모 다운로드 기술 정보

AutoCAD DXF 파일을 상세하게 분석하는 상용화 가능한 Python 솔루션

ezdxf 라이브러리 기반으로 구축된 강력한 DXF 분석 도구로, GUI, 웹, CLI 인터페이스를 모두 지원합니다.

다운로드
📁

DXF 파일을 드래그하거나 클릭하여 업로드

지원 형식: .dxf, .dwg

주요 기능

🔧

자동 DXF 파싱

ezdxf 라이브러리 기반의 강력한 DXF 파일 분석

📊

상세 객체 분석

치수, 원/호, 텍스트, 라인 등 모든 CAD 객체 자동 인식

🎨

레이어 정보 추출

색상, 라인타입, 상태 등 레이어 속성 분석

📈

통계 정보 제공

객체 유형별 개수 및 분포 통계

📝

마크다운 리포트

전문적인 분석 리포트 자동 생성

🖥️

다중 인터페이스

GUI, 웹, CLI 버전 지원

버전별 기능 데모

GUI 버전

tkinter 기반의 데스크톱 애플리케이션

python dxf_analyzer.py --gui
  • 직관적인 사용자 인터페이스
  • 실시간 분석 진행률 표시
  • 탭별 결과 확인
  • 마크다운 리포트 내보내기

웹 버전

Streamlit 기반의 웹 애플리케이션

python dxf_analyzer.py --web
  • 웹 브라우저에서 실행
  • 드래그 앤 드롭 파일 업로드
  • 인터랙티브 차트
  • 실시간 다운로드

CLI 버전

명령줄 인터페이스로 배치 처리 지원

python dxf_analyzer.py --cli file.dxf
  • 스크립트 자동화
  • 배치 처리
  • 서버 환경 최적화
  • 로그 파일 생성
